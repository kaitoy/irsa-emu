# irsa-emu

irsa-emu emulates IAM Roles for Service Accounts (IRSA).

# Build

1. Install Git and Docker.
2. Clone the project.

   ```console
   $ git clone https://github.com/kaitoy/irsa-emu .git
   $ cd irsa-emu
   ```

3. Run Docker build.

   ```console
   $ docker build .
   ```
